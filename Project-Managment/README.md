Project Management Dashboard — Power BI

Overview

This dashboard provides a comprehensive analysis of project performance across different departments, regions, and managers.
It helps decision-makers understand project progress, completion rates, costs, benefits, and overall workload distribution.

The goal of the dashboard is to:
 • Track project completion rate
 • Compare project cost vs. benefit
 • Identify performance by manager
 • Evaluate project complexity
 • Monitor project distribution by region
 • Analyze trends over time

Tools Used
 • Power BI
 • Power Query (Data Cleaning)
 • DAX (Measures & Calculations)
 • Data Modeling
